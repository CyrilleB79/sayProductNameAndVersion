# 程序名称和版本朗读插件 #

* 作者： Patrick ZAJDA <patrick@zajda.fr>
* NVDA 兼容版本：2017.3 或更高
* 下载 [稳定版][1]

此插件添加了一个快捷键来宣布前台应用程序的产品名称和版本，或者将这些信息复制到剪贴板。

* 快捷键：NVDA+Shift+V
* 连按两次可将产品名称和版本复制到剪贴板。
